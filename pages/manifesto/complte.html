# உதவித்தொகை

status: தகுதியுடைய அனைத்து மூத்த குடிமக்களுக்கும் வழங்கப்படும் உதவித்தொகை ரூ.1500 முதல் ரூ.1000 வரை உயர்த்தப்பட்டு, அஞ்சல் துறை மூலம் உதவி வழங்கப்படுவது உறுதி செய்யப்படும்.உயர்த்தப்படவில்லை

# வசதிகள் வழங்கப்படும்

status:அனைத்து மாவட்ட தலைமையகங்களிலும் மூத்த குடிமக்கள் மையங்கள் உருவாக்கப்பட்டு அவர்களுக்கு உணவு, தங்குமிடம் மற்றும் உடை போன்ற அடிப்படை வசதிகள் வழங்கப்படும். கிடைப்பதில்லை .

# முதியோர் பிரிவுகள்

status: அனைத்து அரசு மருத்துவமனைகளிலும் மேம்பட்ட வசதிகளுடன் முதியோர் பிரிவுகள் உருவாக்கப்படும்.நடக்கவில்லை

# வழக்கறிஞர்களுக்கு அலுவலக அறைகள்

status:மாவட்ட நீதிமன்றங்களில் வழக்கறிஞர்களுக்கு அலுவலக அறைகள் கட்டப்படும்.நடக்கவில்லை

# பெண்களுக்கு எதிரான பாலியல் குற்றங்கள்

status: பெண்களுக்கு எதிரான பாலியல் குற்றங்கள் தொடர்பான வழக்குகளை விசாரிக்க தனி நீதிமன்றம் உருவாக்கப்படும், இந்த வழக்குகளில் விரைவாக நீதி வழங்கப்பட வேண்டும்.நடக்கவில்லை

# ரகசியமான முறையில் பராமரிக்கப்படும்.

status:பாலியல் குற்றங்களால் பாதிக்கப்பட்டவர்களின் விவரங்கள் மிகவும் ரகசியமான முறையில் பராமரிக்கப்படும். இந்த விவகாரங்கள் குறித்து விசாரணை நடத்த தனி பிரிவுகள் உருவாக்கப்படும்.தகவல் சரியாக பராமரிப்பதில்லை
